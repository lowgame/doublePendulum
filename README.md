# doublePendulum
 double pendulum with using pygame.

change initial conditions at here (line 13-27):

```
# Pendulum Settings
length1 = 50 # first
length2 = 150 # second
mass1 = 1
mass2 = 1
# Initial Positions
angle1 = np.pi * 1.2
angle2 = np.pi / 3

angular_velocity1 = 0
angular_velocity2 = 0
gravity = 1

# Time step - Like FPS
dt = 0.2
```
equations source: https://web.mit.edu/jorloff/www/chaosTalk/double-pendulum/double-pendulum-en.html
